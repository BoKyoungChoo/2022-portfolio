새로운 것을 배우며 더 나은 사이트를 만들어가는 일이 즐겁습니다.

늘 사용자 입장에서 생각하며 효율적인 UI/UX가 무엇일지 계속 고민하고,
웹 표준과 웹 접근성을 고려한 마크업을 하기 위해 노력합니다.
모든 사람이 사용하기에 편리하고, 즐거운 사이트를 만드는 것이
제가 하는 일이라고 생각합니다.

더 나아가 CSS와 Javascript를 활용한 동적 모션과 서버 통신 후 데이터를 가져와 화면에 나타낼 때 원하던 화면을 구현해 내는 것은 저의 심장을 뛰게 합니다.(💛)

사용자를 배려하는 사이트를 만들어 보람을 얻고,
더 나은 코드를 작성하기 위해 끊임없이 공부하며 발전하고,
많은 시행착오를 겪고 결국엔 이뤄내며 얻는 기쁨과 짜릿함이
제가 개발을 좋아하는 이유입니다.
